## Setup.
To get a local copy of the site:

`git clone https://github.com/ephraim-junior/Delani-Studio.git`

`Extract` the cloned file.

`cd` to Delani-Studio folder.

*Launch* `index.html` file.

## Technologies used.
* HTML5
* CSS3
* Bootstrap
* Javascript
* JQuery
